CSS Typography Cheat Sheet

Tag Name Description
{font-family} Font Family Determines what font an element uses
{font-size} Font Size Sets the size of the font, example in px
{font-weight} Font Weight Affects the boldness of text, the value can be a keyword or a number between 1 to 1000
{font-style} Font Style Typically used to make font italic
{text-align} Text Align Will align text horizontally within an element using common keywords
